# 🌍 GlobalBrief

An AI-powered news summarizer built using React, FastAPI, MongoDB, and Google Gemini.

## Features

- Live News
- AI Summary
- Why It Matters
- Historical Context
- Future Impact
- Timeline
- Daily AI Digest
- Google Authentication
- Bookmarks
- Dark Mode

## Tech Stack

Frontend:
- React
- Bootstrap
- Axios

Backend:
- FastAPI
- MongoDB
- Gemini API
- GNews API
